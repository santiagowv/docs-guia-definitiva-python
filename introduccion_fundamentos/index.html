    Primer programa

    En este programa introducimos la función print() para imprimir datos. 

    print("Hola, este es mi primer programa.")
print("Segunda línea.")
print("Tercer línea.")
print(".............")

    Respuesta: 

    Hola, este es mi primer programa.
Segunda línea.
Tercer línea.
.............

    Variables

    Las variables son una forma de almacenar datos de cualquier tipo.

    Declaración de variables

    • Declaración de nuevas variables: 
      nombre_usuario = "Sebastian"
    • Declaración de variables de tipo constante: 
      NOMBRE_EQUIPO = "mypc"
    • Reasignación de variables: 
      edad_usuario = 12
edad_usuario = edad_usuario + 12

    Reglas de nombramiento

    • Nombramiento de variables:

      • Deben comenzar con una letra o un guion bajo ("-").
      • Pueden contener letras, números y guiones bajos.
      • Python distingue entre mayúsculas y minúsculas, es decir, nombre y Nombre son variables diferentes.

    • Nombres significativos:

      • Nombres descriptivos que reflejen el propósito o contenido de la variable 
            # Evitar
    x = 5

    # Bueno
    edad_usuario = 25
    • Evitar nombres de variables de una letra.
    • Evitar palabras reservadas de Python como: if, while, for, class, etc.
    • Evitar guiones bajos iniciales 
          # Evitar
    _variable_privada = 42

    Tipos de datos

    Son utilizados para representar los diferentes tipos de valores que un programa puede manipular.

    • Enteros (int): Representan números enteros sin decimales. 
          edad = 25
    cuenta_regresiva = -45
    • Flotantes (float): Representan números decimales. 
          pi = 3.14159
    altura = 1.90
    • Cadenas de texto (str): Representan secuencias de caracteres. Pueden contener letras, números y caracteres especiales. 
          nombre = "Juan"
    mensaje = "Hola, ¿Cómo estás?"

    • Booleanos (bool): Representan valores de verdad: True o False. 

          es_mayor_de_edad = True
    esta_lloviendo = False

    • None (None): Representa la ausencia de valor o la falta de un valor válido. 

          resultado = None

    Validar un tipo de dato

    La función type() nos permite validar a que tipo de dato corresponde un valor: 

        edad_usuario = 34
    print(type(edad_usuario))
    Respuesta: 
         <class 'int'>

    Métodos y acciones

    Strings:

    • upper(): Convierte un texto a máyuscula. 
          nombre_usuario = "santiago"
    nombre_usuario = nombre_usuario.upper() # SANTIAGO
    • lower(): Convierte un texto a mínuscula. 
          nombre_usuario = "SANTIAGO"
    nombre_usuario = nombre_usuario.lower() # santiago
    • capitalize(): Convierte en máyuscula la primer letra del texto. 
          mensaje = "hola, bienvenido."
    mensaje = mensaje.capitalize() # Hola, bienvenido.
    • title(): Convierte en máyuscula la primer letra de cada palabra. 
          usuario = "andres felipe"
    usuario = usuario.title() # Andres Felipe
    • count(): Cuenta la cantidad de apariciones de un caracter dentro de un texto. 
          usuario = "andres felipe"
    usuario = usuario.count(e) # 3
    • find(): Regresa la posición inicial de una subcadena de texto dentro de un texto. 
          usuario = "santiago wilches"
    usuario = usuario.find("wilches") # 9
    • replace(): Remplaza una subcadena de texto por otra. 
          mensaje = "Hola, bienvenido."
    mensaje = mensaje.replace("Hola", "Chao") # Chao, bienvenido.
    • strip(): Elimina los espacios en blanco al inicio y al final de un texto. 
          mensaje = "  Hola, bienvenido.  "
    mensaje = mensaje.strip() # Hola, bienvenido.
    • strip(): Elimina los espacios en blanco al inicio y al final de un texto. 
          mensaje = "  Hola, bienvenido.  "
    mensaje = mensaje.strip() # Hola, bienvenido.
    • startswith(): Comprueba si una cadena de texto comienza con una subcadena específica. 
          mensaje = "Hola, bienvenido."
    mensaje = mensaje.startswith("Hola") # True
    • endswith(): Comprueba si una cadena de texto termina con una subcadena específica. 
          mensaje = "Hola, bienvenido."
    mensaje = mensaje.endswith("Hola") # False
    • len(): Regresa la longitud de un objeto. 
          mensaje = "Hola"
    mensaje = len(mensaje) # Hola
    • +: Concatena cadenas de texto. 
          nombre = "Andrew"
    apellido = "Smith
    nombre_completo = nombre + " " + apellido # Andrew Smith
    • *: Concatena el mismo texto n cantidad de veces. 
          nombre = "Andrew"
    nombre = nombre * 4 # AndrewAndrewAndrewAndrew

    Números:

    • abs(): Regresa el valor absoluto de un número. 
          temperatura = -22
    temperatura = abs(temperatura) # 22
    • round(): Aproxima un número con decimales al número entero más cercano. 
          temperatura = 22.3
    temperatura = round(temperatura) # 22

    Conversión de tipos de datos

    • int(): Convertir a entero. 
          numero_string = "2223"
    type(int(2223)) # int
    • str(): Convertir a string. 
          numero = 1234
    type(str(numero)) # str
